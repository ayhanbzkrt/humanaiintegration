# AI'Han Academy — Yapay Zekâ Gelecek Raporu

Yapay zekâ modellerini, yapay zekâ ajanlarını, iş dünyasındaki dönüşümü, yeni becerileri, sorumlu yapay zekâ ilkelerini ve geleceğin teknolojik altyapısını etkileşimli bir web deneyimiyle sunan modern bilgi platformu.

Bu proje; teknik kavramları yalnızca listeleyen statik bir internet sayfası değil, kullanıcıların yapay zekânın güncel gelişim alanlarını keşfedebileceği, bölümler arasında akıcı biçimde gezinebileceği ve güvenilir kaynaklara ulaşabileceği etkileşimli bir dijital rapor olarak tasarlanmıştır.

---

## Proje Hakkında

**AI'Han Academy Yapay Zekâ Gelecek Raporu**, yapay zekâ alanındaki güncel gelişmeleri anlaşılır, görsel ve etkileşimli bir yapı içinde sunar.

Platformun temel amacı; yapay zekâ teknolojileri hakkında farklı bilgi düzeylerine sahip kullanıcıların aşağıdaki konuları bütüncül biçimde inceleyebilmesini sağlamaktır:

* Yeni nesil yapay zekâ modelleri
* Çok modlu yapay zekâ sistemleri
* Yapay zekâ ajanları
* Ajan tabanlı sistemler
* İş dünyasında yapay zekâ kullanımı
* İnsan ve yapay zekâ iş birliği
* Yeni meslekler ve beceriler
* Sorumlu yapay zekâ
* Yapay zekâ etiği ve yönetişimi
* Enerji, donanım ve veri merkezi altyapısı
* Robotik ve fiziksel yapay zekâ
* Geleceğin yapay zekâ teknolojileri

## Kurulum

Projeyi bilgisayarında çalıştırmak için depoyu klonla:

```bash
git clone https://github.com/KULLANICI-ADIN/REPO-ADI.git
```

Proje klasörüne gir:

```bash
cd REPO-ADI
```

Ardından HTML dosyasını tarayıcıda aç:

```bash
open aihan_academy_future_report_arial_tr_terminoloji.html
```

Windows üzerinde:

```bash
start aihan_academy_future_report_arial_tr_terminoloji.html
```

Linux üzerinde:

```bash
xdg-open aihan_academy_future_report_arial_tr_terminoloji.html
```

Dosya doğrudan çift tıklanarak da çalıştırılabilir.

## GitHub Pages ile Yayınlama

Projeyi GitHub Pages üzerinde yayımlamak için:

1. Projeyi GitHub deposuna yükle.
2. Ana HTML dosyasının adını `index.html` yap.

5. `Build and deployment` altında `Deploy from a branch` seçeneğini belirle.
6. Dal olarak `main` seç.
7. Klasör olarak `/root` seç.
8. `Save` düğmesine bas.

## Yazı Tipi ve Türkçe Karakter Desteği

Arayüzde temel yazı tipi olarak Arial kullanılmaktadır.

```css
font-family: Arial, Helvetica, sans-serif;
```

Dosya içinde aşağıdaki karakter kodlaması kullanılmalıdır:

```html
<meta charset="UTF-8">
```

Türkçe metinlerde hatalı büyük harf dönüşümünü önlemek amacıyla, gerekli alanlarda otomatik `uppercase` dönüşümü kullanılmamaktadır.

Özellikle aşağıdaki harf ayrımına dikkat edilmiştir:

```text
i → İ
ı → I
```
